Добрый день! 
Проект VectorsAndMAtrix разработан для создания библиотеки базовых арифметических операций
над векторами и матрицами.

Файл Vector.cs хранит в себе класс Vector c необходимыми полями и методами, которые предназначены
для работы с векторами n-мерной размерности. Реализованы следующие методы(операции):
1. Вычисление длины вектора.
2. Умножение вектора на число.
3. Сложение векторов.
4. Вычитание векторов.
5. Скалярное произведение векторов.
6. Векторное произведение векторов.
7. Смешанное произведение векторов.

Файл Matrix.cs хранит в себе класс Matrix c необходимыми полями и методами, которые предназначены
для работы с матрицами nxn. Реализованы следующие методы(операции):
1. Умножение матрицы на число.
2. Сложение матриц.
3. Нахождение определителя матрицы.
4. Умножение вектора на матрицу.
5. Умножение матрицы на вектор.
6.Умножение матрицы на матрицу.
7. Транспонирование матрицы.
8. Нахождение обратной матрицы с помощью метода Гаусса.

View.cs реализует визуализацию консольного окна и вызов арифметических операций.
Verification.cs хранит в себе методы, необходимые для проверки условий выполнения операций.
InitialConditions.cs, в свою очередь, представляет собой класс с начальными условиями.
Примечание. Изменение начальных условий производится вручную пользователем.
CholetskyDecomposition.cs хранит в себе реализацию обращения симметрической матрицы с использованием разложения Холецкого.
